### react-challenge

## Installation

Use the package manager [npm](https://www.npmjs.com/get-npm) or [yarn](https://yarnpkg.com/lang/en/docs/install/#debian-stable) to install react-challenge.

```bash
npm install
```

or

```bash
yarn
```
